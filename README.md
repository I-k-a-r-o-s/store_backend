# Store Backend API

A RESTful API backend for an e-commerce store built with **Node.js**, **Express.js**, and **MongoDB**. This server provides endpoints to manage product data with full CRUD (Create, Read, Update, Delete) operations.

## âœ¨ Features

- **Create Products**: Add new products with name, price, and image URL
- **Read Products**: Retrieve all products from the database
- **Update Products**: Modify existing product information
- **Delete Products**: Remove products from the database
- **Error Handling**: Comprehensive error responses with meaningful messages
- **Data Validation**: Required field validation before database operations
- **Timestamps**: Automatic creation and modification timestamps for all products
- **Database Connection**: Secure MongoDB connection with environment variables
